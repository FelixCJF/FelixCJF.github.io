Java面试题总结(上)

网上找的题目，自己一题一题做的，查缺补漏

一、Java 基本概念

1. Java 语言的优点?

简单、高效
跨平台：生成字节码文件后，可以跨平台运行
交互式特性:面向网络编程
面向对象 ：除了基本数据类型，在Java语言中都是以类的形式定义
动态内存管理 :Java使用自动垃圾回收机制来进行内存管理
多线程机制：支持多线程，是Java能够并发处理多项任务
安全性 ：使用实时内存分配

2.什么是 Java 虚拟机？为什么 Java 被称作是“平台无关的编程语言”？

JVM是一个可执行Java字节码的虚拟进程
Java程序可以运行在任意平台

3. Java 和 C++ 有何区别?

都是面向对象语言
Java使用自动内存管理，c++需要程序员分配内存和释放内存
Java没有指针，c++有指针
Java异常机制可以捕获例外情况，增加了容错能力
Java中含字符串，而C++中没有
Java只能单继承，而C++可以多重继承
Java中goto是保留字

4. JDK 和 JRE 的区别是什么？

JDK：Java程序开发的最小环境
JRE：Java程序运行的标准环境
JDK包括JRE

5. Java 支持的数据类型有哪些？什么是自动拆装箱？

基本数据类型 ：char,byte,short,int,float,long,double,boolen
引用数据类型 ：
自动装箱就是把基本数据类型转换成引用数据类型，比如int转换成Integer
自动拆箱就是把引用数据类型转换成基本数据类型，比如Double转换成Double

6. 什么是值传递和引用传递？

值传递，也就是对象的传递，传递的是一个对象的副本，传递的副本对象发生改变，不会对源对象产生影响
引用传递，也就是传递对象的引用，外部对引用对象所做的改变会放映到所有的对象中

7. 一个 “.java” 源文件中是否可以包括多个类（不是内部类）？有什么限制？

可以包含多个类，当只能有一个public修饰的类，切public类名必须和文件名一致

8. 静态变量和实例变量的区别？

静态变量被static修饰，实例变量不被static修饰
静态变量属于类，实例变量属于对象
静态变量存在于方法区中，实例变量存在于堆中
静态变量随着类的加载而加载，随着类的消失而消失，实例变量随着对象的创建而存在，随着对象的回收而消失
静态变量既可以被对象调用，也可以被类名调用，实例变量只能被对象调用

9. 不通过构造函数也能创建对象吗?

可以
通过new创建一个对象
通过反射
通过调用对象的clone方法
通过反序列化手段，Java.io.ObjectStream的readObject方法

后两个不通过构造函数创建对象

10. 是否可以从一个静态（static）方法内部发出对非静态（non-static）方法的调用？

不能，
静态方法只能访问静态成员，因为静态方法优先于对象的创建，所以当调用非静态方法的时候，对象可能还没有创建

11. 如何实现对象克隆？

两种方法
实现Cloneable，并重写Object中的clone方法
实现Serializable，并通过对象的序列化和反序列化实现正真的深度克隆

14. Anonymous Inner Class(匿名内部类)是否可以继承其它类？是否可以实现接口？

定义匿名内部类的前提是：必须继承一个类或者实现一个接口

12. 内部类可以引用它的包含类（外部类）的成员吗？有没有什么限制？

可以，包括私有成员

13. 列出自己常用的 jdk 包.

java.lang 含有系统基础类，唯一一个不需要导入就可以使用的,比如String
java.net 含有与网络有关的类 比如URL
Java.io 含有与输入输出有关的类 比如文件操作 File,InputStream,OutputStream
java.sql 含有与数据库有关的类 比如Connection，Statement，ResultSet
java.util 含有Java相关的工具类 比如集合相关，Concention，List,Set,Map

14. JDK, JRE 和 JVM 的区别？

JDK: Java程序开发的最小环境
JRE：Java程序运行的标准环境
JVM：将字节码文件转换为机器码文件，Java语言的核心
JDK包括JRE。JRE包括JVM

二、面向对象编程

1. Java 中的方法覆盖 (Overriding) 和方法重载 (Overloading) 是什么意思？

方法重载：方法名相同，参数列表不同。与返回值无关
方法覆盖：方法签名一样，返回值一样。子类覆盖父类时，保证子类的访问权限大于父类，子类只能抛出父类异常或异常的子类，并且被覆盖的方法不能private修饰

2. Overload 和 Override 的区别？ Overloaded 的方法是否可以改变返回值的类型?

重载是在同一个类中，覆盖是在子类中覆盖父类中的方法，重载和返回值无关，覆写需要保证方法签名和返回值都一样

3. Java 中，什么是构造函数？什么是构造函数重载？什么是复制构造函数？

构造函数：每一个对象被创建时都会调用构造函数，当没有给类提供构造函数的时候，Java编译器会创建一个默认的构造函数
构造函数重载：一个类中定义多个构造函数，每一个构造函数都有不同的参数列表
复制构造函数：C++里面的

4. 构造器 Constructor 是否可被 Override?

不可以，覆写的前提是继承，所以构造器只能被重载，不能被覆写

5. Java 支持多继承么？

不支持，但Java支持多实现

6. 接口和抽象类的区别是什么？

抽象类只能被继承，接口只能被实现，但可以多实现。
抽象类中可以有非抽象方法，接口中只能含有抽象方法，需要被子类实现
接口中成员函数修饰符是public，抽象类的成员修饰符可以自定义
接口中的变量默认final，抽象类可以声明非final变量
接口是绝对抽象的不能实例化，抽象类也不可以被实例化，但有构造函数

7. 下列说法正确的有（）

A．class 中的 constructor 不可省略
B．constructor 必须与 class 同名，但方法不能与class同名
C．constructor在一个对象被 new 时执行
D．一个 class 只能定义一个 constructor
答案：C
解析：普通的类方法是可以和类名同名的，和构造方法唯一的区分就是，构造方法没有返回值。

8. Java 接口的修饰符可以为?

A. private
B. protected
C. final
D. abstrac
答案：CD
接口的属性默认是 public static final常量，且必须赋初值。
接口的方法默认是 public abstract

9. 下面是 People 和 Child 类的定义和构造方法，每个构造方法都输出编号。在执行 new Child(“mike”) 的时候都有哪些构造方法被顺序调用？请选择输出结果

class People {
String name;
public People() {
System.out.print(1);
}
public People(String name) {
System.out.print(2);
this.name = name;
}
}
class Child extends People {
People father;
public Child(String name) {
System.out.print(3);
this.name = name;
father = new People(name + ":F");
}
public Child() {
System.out.print(4);
}
}

A. 312
B. 32
C. 432
D. 132
答案：D
必须先调用父类构造函数，没有显式调用的话，则默认调用默认的父类构造参数

10. 两个对象值相同(x.equals(y) == true)，但却可有不同的 hash code，这句话对不对？

不对，如果两个对象值相同，则它们hashcode相同 ，如果它们hashcode相同，值不一定相同

11. 接口是否可继承（extends）接口? 抽象类是否可实现（implements）接口? 抽象类是否可继承具体类（concrete class）?

可以，抽象类可以继承具体类，前提是抽象类必须有明确的构造函数

12. 指出下面程序的运行结果:

class A{
static{
System.out.print("1");
}
public A(){
System.out.print("2");
}
}
class B extends A{
static{
System.out.print("a");
}
public B(){
System.out.print("b");
}
}
public class Hello{
public static void main(String[] args){
A ab = new B();
ab = new B();
}
}

答：执行结果：1a2b2b。
创建对象时构造器的调用顺序是：先初始化静态成员，然后调用父类构造器，再初始化非静态成员，最后调用自身构造器

13. Class.forName（String className）这个方法的作用

通过类全名获取类对象

14. 什么是 AOP 和 OOP，IOC 和 DI 有什么不同?

AOP: 面向切面编程
OOP：面向对象编程
IOC：控制反转
DI： 依赖注入

三、关键字

1. ”static” 关键字是什么意思？Java 中是否可以覆盖(override) 一个 private或者是 static 的方法？

“static” 关键字表明一个成员变量或者是成员方法可以在没有所属的类的实例变量的情况下被访问。
不可以

2. 是否可以在 static 环境中访问非 static 变量？

不可以，static变量是属于类的，调用非static变量时，对象可能还没有被创建

3. volatile关键字是否能保证线程安全？

不能,保证线程的可见性，可以正确读取其他线程写入的值

4. Java 有没有 goto?

作为保留字

5. Java 中的 final关键字有哪些用法

修饰类：不能被继承
修饰变量：变量值不能改变
修饰方法：不能被覆写

6. 什么时候用 assert？

调试的时候

7. final, finally, finalize 的区别?

final :修饰词，修饰类，变量，方法
finally：关键字，异常捕获的时候使用，无论是否发生异常，都会执行
finalize：Object中的方法，垃圾回收器进行垃圾清理之前调用finalize（）做必要的清理工作

四、基本类型和运算

1. 说说 & 和 && 的区别。

&：与操作 两边表达式都为TRUE时，才是TRUE，只要一边为FALSE，则最后结果为FALSE
&&：短路与 左边的表达式错误，则不执行右边的表达式

2. 用最有效率的方法算出 2 乘以 8 等於几?

2<<3

3. 存在使 i + 1 < i的数吗

存在，当i为int的最大数时，加一则变成了负数，等式成立

不正确

8. int 和 Integer 有什么区别?

int 是基本数据类型
Integer 是包装类型

9. char 型变量中能不能存贮一个中文汉字?为什么?

可以，char占两个字节

10. Math.round(11.5) ? 等于多少? Math.round(-11.5)等于多少?

返回与数值最接近的长整数

五、字符串与数组

1. 下面程序的运行结果是（）

String str1 = "hello";
String str2 = "he" + new String("llo");
System.err.println(str1 == str2)

false ==判断对象的地址

import java.io.*;
import java.util.*;
public class foo{
public static void main (String[] args){
String s;
System.out.println("s=" + s);

答案：C
java中所有定义的基本类型和对象都必须初始化之后才能输出对象

3. String 是最基本的数据类型吗?

不是，除了基本类型和枚举类型，其他都是引用类型

4. 数组有没有 length() 方法? String 有没有 length() 方法？

数组中有length属性，String中有length（）方法

5. 是否可以继承 String 类?

String是final类，不可以被继承

6. String 和StringBuilder、StringBuffer 的区别?

String：只读字符串，长度固定，且不可变
StringBuilder：字符缓冲区，长度可变，线程安全
StringBuffer：长度可变、效率高，线程不安全

7. String s=new String(“xyz”);创建了几个字符串对象？

两个，一个是常量池中的“xyz”，另一个是堆上的对象s

答：
String s=”12345″; long num=Long.valueOf(s).longValue();

原始内容没变，只是s不指向“hello”而是指向一个新的对象

14. 什么是 Java 序列化，如何实现 Java 序列化？

序列化：处理对象流的机制，将对象流化
序列化的实现：将需要被序列化的类实现 Serializable 接口，该接口没有需要实现的方法， implements Serializable 只是为了标注该对象是可被序列化的，然后使用一个输出流(如：FileOutputStream)来构造一个ObjectOutputStream(对象流)对象，接着，使用ObjectOutputStream对象的writeObject(Object obj）方法就可以将参数为obj的对象写出(即保存其状态)，要恢复的话则用输入流
