Java中的网络编程

Java
  • Java中的网络编程

有关Java网络编程的笔记整理

网络通信三要素：IP地址，端口，还有传输协议

IP地址

一个网络设备标识
Java中获取IP地址：

//导包
import java.net.*;
public class IPDemo{
public static void main(String[] args)throws Exception{
    //获取InetAddress对象
    InetAddress ip = InetAddress.getByName("www.baidu.com");
    //然后调用InetAddress对象中的getHostAddress()方法获得IP
    //输出到控制台上看看结果吧：
    System.out.println("www.baidu.com的IP地址是："+ip.getHostAddress());
}
}

端口

用来标记进程的逻辑地址
1、端口的范围从0~65535（2的16次方减1）

2、系统保留端口为：0~1024

3、一些常用的端口，FTP：21，SMTP：25，HTTP：80

传输协议：TCP、UPD

UPD协议

UPD协议特点：

  1. 将数据封装为数据包，面向无连接
  2. 每个包大小限制在64kb
  3. 因为面向无连接，所以速度快，不可靠
  4. 有发送端与接收端

发送端的操作：
1、建立Socket服务，随便你指不指定端口，不指定系统分配。另外就是端口顺延，如果这次端口是1555，下次就是1556，因为当你运行完程序后，不能保证端口也释放了，为了防止出错。

2、将要发送的数据封装到包中。

3、发送数据，通过send方法发送出去，管它有没有收到呢。

4、关闭资源。 

//发送端
//导包
import java.net.*;
public class UPDDemo{
public static void main(String[] args)throws Exception{

//创建Socket服务
DatagramSocket ds = new DatagramSocket();
//不指定端口也行，稍后在数据里指定。
//把数据打包
String data = "UPD传输演示";
byte[] buf = data.getBytes();
//把数据转换成字节数组
DatagramPacket dp = new DatagramPacket(buf,
                          buf.length,
                          InetAddress.getByName("10.1.31.127"),
                          10000);//通过名字获取IP，并指定端口

//发送数据
ds.send(dp);
//关闭资源
ds.close();
}
}

UPD接收端的操作：
1、建立服务

2、定义一个缓冲区来接受数据

3、调用DatagramPacket中的各种方法来达到我们的需求

4、关闭资源

//接收端
//导包
import java.net.*;
public class UPDServer{
public static void main(String[] args)throws Exception{

//建立服务监视这个端口
DatagramSocket ds = new DatagramSocket(10000);

//定义一个缓冲区用于接收数据
byte[] buf = new byte[1024];//可以乘以64，因为一个包在64K以内。
DatagramPacket dp = new DatagramPacket(buf,buf.length);
ds.receive(dp);

//获取IP，这是一个习惯，因为要知道这数据是从那里来的
String ip = dp.getAddress().getHostAddress();
System.out.println("来自"+ip+"的数据是");

//把数据弄出来吧
System.out.println(new String(buf,0,dp.getLength()));

//关闭资源
ds.close();        
}
}

TCP协议

tcp协议特点：

  1. 基于IO流进行数据传输的，面向连接
  2. 数据传输没有大小限制
  3. 面向连接通过三次握手，四次挥手机制保证数据的完整性和可靠性
  4. 面向连接，速度慢
  5. 分为服务端和客户端

客户端的操作步骤：

1、建立服务Socket

2、获取输出流，把数据变成字节数组， 通过输出流发送给服务端。

3、关闭输出流，获取输入流，获取反馈信息

4、关闭资源

//导包
import java.io.*;
import java.net.*;

public class TCPClient {

public static void main(String[] args) {

// 创建服务
Socket s = null;
try {
    if (s == null)
        s = new Socket("10.1.31.127", 13000);
    //把数据转换成字节数组
    byte[] buf = "TCP传输演示".getBytes();

    //获取输出流
    OutputStream out = s.getOutputStream();
    //发送数据
    out.write(buf);

    //关闭发送流
    s.shutdownOutput();

    //获取输入流，获取反馈信息
    InputStream in = s.getInputStream();
    byte[] buffer  = new byte[1024];
    int len = in.read(buffer);

    //打印反馈信息
    System.out.println(new String(buffer,0,len));
    } catch (Exception e) {
    System.out.println(e.toString());
    }finally{
    if(s!=null){
        try{
            s.close();
        }catch(Exception e)
        {
            System.out.println(e.toString());}
        }
    }

}
}

服务端操作步骤：
1、建立服务ServerSocket服务，然后。用ServerSocket的accept（）方法得到Socket服务

2、获取输入流，然后可以得到数据

3、对读到的数据进行处理

4、反馈信息给客户端

5、各种关闭资源

    //导包
import java.io.*;
import java.net.*;

public class TCPServer {

public static void main(String[] args)throws Exception {

    //创建服务
    ServerSocket ss = new ServerSocket(13000);

    Socket s = ss.accept();
    //先搞到IP地址
    String ip = s.getInetAddress().getHostAddress();

    //输出链接上来的机器
    System.out.println(ip+"-----connected");

    //获取输入流
    InputStream in  = s.getInputStream();
    //读取数据
    byte[] buf = new byte[1024];

    int len = 0;
    while((len = in.read(buf))!=-1){
        //打印到控制台上吧
        System.out.println(new String(buf,0,len));
    }
    s.shutdownInput();

    //发送反馈信息

    OutputStream out = s.getOutputStream();

    out.write("服务端收到啦".getBytes());

    //关闭资源
    s.close();
    ss.close();
}

}
